# Honda Replica Website

This is a simple replica website for Honda cars. It showcases multiple full-screen sections with car models and a fixed navigation menu,

---

## Features

- **Responsive Sections**: Full-screen background images with overlay text panels for each car model.
- **Fixed Navigation Menu**: Top menu stays visible while scrolling.
- **Canvas Logo**: Custom Honda-style logo drawn using HTML5 Canvas.
- **Footer**: Includes an SVG logo and informative text.
- **Responsive Design**: Adjusts layout and text size for smaller screens.

---

## Technologies Used

- HTML
- CSS
- JavaScript
- [SweetAlert2](https://sweetalert2.github.io/) for popups
- Google Fonts: Montserrat & Oswald
