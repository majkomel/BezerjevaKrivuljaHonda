# Honda replika spletne strani

To je preprosta replika spletne strani za avtomobile Honda. Prikazuje več celozaslonskih sekcij z modeli avtomobilov ter fiksni navigacijski meni.

---

## Funkcionalnosti

- **Odzivne sekcije**: Celozaslonske slike ozadja s prekrivnimi besedilnimi paneli za vsak model avtomobila.
- **Fiksni navigacijski meni**: Zgornji meni ostane viden med pomikanjem po strani.
- **Canvas logotip**: Logotip v slogu Honde, narisan s pomočjo HTML5 Canvas.
- **Noga strani (footer)**: Vsebuje SVG logotip in informativno besedilo.
- **Odzivno oblikovanje**: Prilagajanje postavitve in velikosti besedila za manjše zaslone.

---

## Uporabljene tehnologije

- HTML
- CSS
- JavaScript
- [SweetAlert2](https://sweetalert2.github.io/) za pojavna okna
- Google Fonts: Montserrat & Oswald
